# Our Hangout 개발 순서 (앱 + 백엔드)

## 0. 현재 상태
- [x] Android 앱 기본 UI/채팅 화면 동작
- [x] 이미지/동영상 선택 UI
- [x] 기기 언어 기반 앱 내부 텍스트 한국어/영어 분기
- [x] Android 앱 이름 지역화 (`ko`: `우리들의아지트`, 기본: `Our Hangout`)

## 1. 제품/정책 먼저 고정 (필수)
1. 서비스 정책 정의
- [ ] 연령 정책(아동 계정 허용 방식, 보호자 동의 방식)
- [ ] 신고/차단/제재 정책
- [ ] 데이터 보관 기간(메시지, 미디어, 로그)

2. 개인정보/보안 범위 정의
- [ ] 필수 수집 정보 최소화
- [ ] 개인정보 처리방침/이용약관 초안
- [ ] 국가별 법규 체크(아동 관련 규정 포함)

## 2. 계정/인증 (앱 공통 기반)
1. 가입/로그인
- [ ] 휴대폰 번호 OTP 또는 이메일 OTP
- [ ] 닉네임/프로필 사진/초기 설정
- [ ] JWT(Access/Refresh) + 토큰 재발급

2. 기기 보안
- [ ] 로그인 세션 관리(다중 기기, 강제 로그아웃)
- [ ] 루팅/무결성 체크(선택)
- [ ] 앱 잠금(PIN/생체인증)

## 3. 친구 시스템 (메신저 핵심)
1. 친구 추가/조회
- [ ] 친구 ID 검색
- [ ] 친구 요청/수락/거절
- [ ] 친구 목록/차단 목록

2. 프라이버시
- [ ] 전화번호 비노출 옵션
- [ ] 친구추천 끄기 옵션
- [ ] 마지막 접속/읽음 공개 범위

## 4. 채팅 백엔드 MVP
1. 실시간 통신
- [ ] WebSocket 서버(연결/재연결/헬스체크)
- [ ] 1:1 채팅방 생성/목록/입장
- [ ] 메시지 송수신 ACK(전송됨/수신됨/읽음)

2. 저장소 모델
- [ ] User, Friend, ChatRoom, Message, DeviceToken 스키마
- [ ] 메시지 페이지네이션(최근 N개 + 이전 불러오기)
- [ ] 오프라인 큐(재전송)

## 5. 앱 기능 MVP
1. 채팅 UX
- [ ] 실제 서버 연동(현재 더미 메시지 제거)
- [ ] 읽음 상태/타이핑 상태
- [ ] 메시지 삭제(나만/전체)

2. 미디어
- [ ] 이미지/동영상 업로드 API 연동
- [ ] 썸네일/압축/업로드 진행률
- [ ] 다운로드 권한/캐시 정리

## 6. 안전 기능 (아동 친화 우선)
1. 기본 안전장치
- [ ] URL 클릭 차단(현재 UI 문구 -> 실제 차단 로직 확장)
- [ ] 금칙어/유해어 필터
- [ ] 신고/차단 1탭 처리

2. 보호자 옵션(권장)
- [ ] 보호자 연결 코드
- [ ] 친구 추가 승인 모드
- [ ] 이용시간 제한/야간 제한(선택)

## 7. 알림/백그라운드
1. 푸시
- [ ] FCM 토큰 등록/갱신
- [ ] 채팅/친구요청/시스템 알림 타입 분리
- [ ] 방해금지 시간대

2. 안정성
- [ ] 앱 종료/재부팅 후 자동 복구
- [ ] 네트워크 변경(와이파이<->LTE) 복구 테스트

## 8. 그룹/아지트 기능 (다음 단계)
1. 아지트(그룹)
- [ ] 그룹 생성/초대/권한(방장/관리자)
- [ ] 공지/고정 메시지
- [ ] 그룹 프로필/배너

2. 개인 방
- [ ] 개인 꾸미기 테마/배경/배지
- [ ] 방별 권한/초대 링크(필요 시 제한형)

## 9. 운영 준비
1. 백엔드 인프라
- [ ] API 서버, WebSocket 서버, DB, 오브젝트 스토리지, Redis
- [ ] TLS(HTTPS/WSS), 비밀키 관리, 백업/복구
- [ ] 모니터링(오류율, 지연, 연결수)

2. 배포 파이프라인
- [ ] dev/stage/prod 환경 분리
- [ ] 마이그레이션 자동화
- [ ] 앱 서명키/릴리즈 관리

## 10. QA/출시
1. 테스트
- [ ] 단위 테스트(핵심 도메인)
- [ ] 통합 테스트(인증, 친구, 채팅)
- [ ] E2E 테스트(실기기 다중 계정)

2. 출시
- [ ] 내부 테스트 트랙 배포
- [ ] 크래시/ANR 모니터링
- [ ] 베타 피드백 반영 후 정식 배포

---

## 권장 구현 우선순위 (빠른 MVP)
1. 인증(OTP) + 프로필
2. 친구 요청/목록
3. 1:1 실시간 채팅 텍스트
4. 읽음/재전송/오프라인 복구
5. 이미지 전송
6. 신고/차단
7. 푸시 알림
8. 그룹(아지트)
9. 개인방 꾸미기

## 백엔드 최소 기술 구성 (권장 예시)
- API: Node.js (NestJS) 또는 Go
- 실시간: WebSocket Gateway
- DB: PostgreSQL
- 캐시/큐: Redis
- 미디어 저장: S3 호환 스토리지
- 푸시: Firebase Cloud Messaging
