# Cyworld Personal Hideout Concept

## 1. 이 탭의 정체성

이 탭은 그룹 아지트가 아니라,

`싸이월드 미니홈피 감성의 나만의 공간`

이어야 합니다.

중심은 방 자체보다 `나를 보여주는 연출`입니다.

즉, 이 탭의 핵심은:

- 미니룸
- 미니미 또는 대표 아바타
- 상태메시지
- BGM
- 방명록
- 사진첩
- 다이어리
- 방문자 흔적

입니다.

게임처럼 방을 직접 조작하는 느낌보다,

`꾸며진 내 공간을 보여주고 방문받는 느낌`

이 더 중요합니다.

## 2. 탭 구조 제안

현재 아지트 개념을 두 층으로 나누는 것이 자연스럽습니다.

### 상위 구분

1. `내 공간`
2. `우리 아지트`

역할:

- `내 공간`: 싸이월드풍 미니홈피 + 미니룸
- `우리 아지트`: 가족/친구 그룹 아지트

이렇게 하면 개인 정체성과 그룹 운영 기능이 섞이지 않습니다.

## 3. 내 공간 탭 IA

### 첫 화면 구조

1. 미니홈피 헤더
2. 미니룸 대표 장면
3. 빠른 메뉴
4. 콘텐츠 탭
5. 방문자/일촌 영역

### 헤더 구성

- 이름
- 상태메시지
- Today / Total 방문자
- BGM 플레이어
- 대표 썸네일 또는 미니미

헤더는 "이 공간의 주인이 누구인지"를 한 번에 전달해야 합니다.

### 메인 히어로: 미니룸

미니룸은 전체 화면 방이 아니라,

`미니홈피 메인 위젯`

처럼 보여야 합니다.

권장 구성:

- 작은 2.5D 방 한 칸
- 대표 가구 3~6개
- 테마 배경
- 미니미 또는 캐릭터
- 계절 소품

중요한 점:

- 복잡한 조작 UI는 숨기고
- 홈에서는 결과물만 예쁘게 보여준다

## 4. 콘텐츠 탭 제안

싸이월드 감성은 단순히 방이 아니라 "콘텐츠 흔적"에서 완성됩니다.

권장 탭:

1. `방명록`
2. `사진첩`
3. `다이어리`
4. `캘린더`

### 방명록

- 친구가 짧은 글 남기기
- 최근 순 정렬
- 스티커 반응 가능

### 사진첩

- 대표 사진 4~9장 그리드
- 추억 중심
- 캡션 짧게

### 다이어리

- 감정 기록
- 오늘의 한마디
- 공개 범위 옵션

### 캘린더

- 생일
- 약속
- 특별한 날 기록

## 5. 미니룸 편집 UX

홈 화면은 미니홈피처럼 보여주고,
편집 화면만 별도로 열어야 합니다.

### 편집 화면 목적

- 내 미니룸의 테마를 바꾼다
- 가구/소품을 배치한다
- 대표 한 컷을 만든다

### 편집 화면 구성

상단:

- 뒤로
- 방 이름
- 저장

중앙:

- 미니룸 캔버스

하단:

- 카테고리 탭
  - 가구
  - 소품
  - 벽지
  - 바닥
  - 스티커

오른쪽 또는 플로팅:

- 회전
- 앞으로/뒤로
- 삭제

핵심은:

- 편집은 쉬워야 하지만
- 홈에서는 편집 UI가 보이면 안 된다

## 6. 시각 톤

권장 톤:

- 파스텔 블루
- 크림 화이트
- 연핑크
- 민트
- 노트/종이 같은 질감

시각적 장치:

- 도트 패턴
- 스티커 테이프
- 폴라로이드 프레임
- 둥근 탭
- 작은 아이콘 장식

피해야 할 것:

- 너무 현대적인 SNS 카드 UI
- 과한 게임 HUD
- 가족 대시보드 같은 실무형 화면
- 다크하고 무거운 분위기

## 7. 왜 이전 방향과 달라야 하는가

이전의 "방 한 칸 게임" 방향은

- 장면은 예쁘지만
- 싸이월드 특유의 개인 기록성과 방문 문화가 약합니다

싸이월드풍 공간은:

- 방만 예쁜 것이 아니라
- 기록과 흔적이 같이 보여야 합니다

그래서 `미니룸 + 방명록 + BGM + 사진첩`이 한 화면 안에 같이 살아야 합니다.

## 8. 사용자 흐름

### Flow A. 내 공간 보기

`내 공간 탭 진입 -> 미니룸과 상태메시지 확인 -> 최근 방명록 확인`

### Flow B. 내 공간 꾸미기

`내 공간 -> 미니룸 꾸미기 -> 테마 변경/소품 배치 -> 저장 -> 홈에 반영`

### Flow C. 친구 방문

`친구 프로필 -> 친구 공간 방문 -> 방명록 남기기 -> 사진첩 보기`

## 9. 우리 아지트와의 관계

권장 구조:

- `내 공간`: 감성, 자기표현, 방문 문화
- `우리 아지트`: 가족/친구 그룹 운영

즉,

- 내 공간은 `미니홈피`
- 우리 아지트는 `공유 베이스`

입니다.

같은 탭 안에서 세그먼트로 분리하면 가장 자연스럽습니다.

## 10. 1차 MVP

최소 범위:

- 내 공간 홈
- 미니룸 대표 장면
- 상태메시지
- BGM 표시
- 방명록 목록
- 사진첩 미리보기
- 미니룸 간단 편집

제외:

- 실시간 방 방문 애니메이션
- 아이템 거래
- 복잡한 게임형 경제 시스템

## 12. 구현 해석

현재 앱 기준으로는 이 순서가 적절합니다.

1. 아지트 탭을 `내 공간 / 우리 아지트` 세그먼트로 분리
2. `내 공간`은 싸이월드풍 홈 UI로 구성
3. 그룹 기능은 `우리 아지트`로 이동
4. 미니룸은 홈의 일부 컴포넌트로 보이게 구성
5. 방명록/사진첩/다이어리 탭을 추가

핵심은:

`개인 공간은 방 꾸미기 게임이 아니라, 꾸며진 미니홈피여야 한다`

는 점입니다.
